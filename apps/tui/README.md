# FreeCode TUI

Terminal UI that drives ChatGPT via Playwright/CDP.

## Setup

```sh
cd apps/tui
pnpm build
npm link
```

## Run

```sh
freecode
```

## Development

```sh
cd apps/tui
pnpm dev
```

## Usage
this is the command for arch linux - 
```sh
chromium --remote-debugging-port=9222
```

- use /freecode <text>
